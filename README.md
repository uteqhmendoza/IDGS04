# Foobar

Esta es una librería de JavaScript para hacer triángulos en consola.

## Installation

Para la instalación utiliza npm

```bash
npm i triangulus
```

## Usage

```console
piramide

# returns 
    *    
   * *   
  * * *  
 * * * *  
* * * * *  
 * * * *   
  * * *    
   * *     
    * 
piramide(5);

```
